# 👗 Digital Wardrobe Project  

Інтелектуальна система автоматичної інвентаризації гардероба, яка перетворює відео-обхід гардероба на структуровану базу даних із візуальними представленнями речей.
Рішення побудоване як багаторівневий pipeline обробки візуальних даних, що поєднує AI-аналітику та автоматизовану обробку зображень для створення цифрового гардероба користувача.

## 📂 Структура проєкту
```
project-root/
│
├── .env                  # Конфігурація середовища (API ключі)
├── .gitignore           # Виключення для Git
├── main.py              # Основний orchestrator pipeline
├── requirements.txt     # Залежності проєкту
│
├── pipeline/            # Основна логіка обробки
│   ├── schema.py        # Структура та стандарти даних
│   ├── video_utils.py   # Утиліти для роботи з відео та зображеннями
│   └── prompts.py       # AI-інструкції та конфігурація
│
└── data/                # Робочі дані (локально)
    ├── videos/          # Вхідні відео
    ├── frames/          # Проміжні зображення
    ├── processed/       # Оброблені зображення
    └── final/           # Фінальні результати
```
## ⚙️ Загальний Pipeline

1. **Video Analysis**  
   Аналіз відео для визначення предметів гардероба та їх основних характеристик.

2. **Frame Selection**  
   Вибір найбільш репрезентативного кадру для кожного предмета.

3. **Image Processing**  
   Базова обробка зображення для підготовки до фінального вигляду.

4. **Final Image Generation**  
   Формування фінального зображення у студійному стилі з урахуванням контексту користувача.

## 📁 Призначення файлів

### `pipeline/schema.py`  
Визначає структуру та стандарти даних у системі. Забезпечує узгодженість між AI-моделями, бекендом і базою даних.

- Містить довідники (типи, кольори, матеріали тощо)
- Визначає формат відповіді для AI
- Забезпечує консистентність даних у всьому pipeline

---

### `pipeline/prompts.py`  
Містить інструкції для AI-моделей.

- Описує ролі AI у системі (аналіз, обробка, генерація)
- Визначає очікувану поведінку моделей
- Забезпечує стабільність результатів

---

### `pipeline/video_utils.py`  
Містить утиліти для обробки відео та зображень.

- Робота з відео
- Обробка кадрів
- Підготовка зображень
- Генерація фінального результату

---

### `main.py`  
Головний orchestrator системи.

- Керує виконанням pipeline
- Об’єднує всі компоненти
- Обробляє відео та формує результати
